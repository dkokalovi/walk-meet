##  Opis aplikacije

**walk&meet** je web aplikacija za upoznavanje novih ljudi kroz zajedničke šetnje.

Aplikacija omogućuje:
- registraciju i prijavu korisnika
- pregled drugih korisnika
- filtriranje korisnika (npr. po spolu)
- pregled profila korisnika
- slanje likeova
- slanje poruka
- pregled poruka i obavijesti
- slanje prijedloga za šetnju (datum, vrijeme, lokacija)
- uređivanje vlastitog profila

Cilj aplikacije je omogućiti korisnicima prirodniji način upoznavanja kroz aktivnosti u stvarnom svijetu.
